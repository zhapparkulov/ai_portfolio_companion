# `.codex/` — рабочая память Codex

Эта папка нужна, чтобы Codex работал в проекте так же предсказуемо, как Claude:
сначала читал правила, затем делал изменения и оставлял короткий журнал того,
что было изменено.

Файлы:

- [`CODEX.md`](./CODEX.md) — правила работы Codex внутри проекта.
- [`changes.md`](./changes.md) — журнал изменений, решений и проверок.
- [`change-template.md`](./change-template.md) — шаблон записи для новых сессий.

Источник правды по архитектуре остается здесь:
[`../docs/ARCHITECTURE.md`](../docs/ARCHITECTURE.md).

Рекомендуемый порядок чтения для Codex:

1. `docs/ARCHITECTURE.md` — архитектура проекта.
2. `.claude/CLAUDE.md` — общие правила проекта.
3. `.codex/CODEX.md` — как именно Codex должен работать и вести журнал.
4. `.codex/changes.md` — что уже менялось ранее.
