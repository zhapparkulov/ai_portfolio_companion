# `.claude/` — AI-пайплайн разработки

Эта папка делает проект готовым к работе с AI. Любой AI-ассистент, открывающий этот репозиторий,
должен сначала прочитать эти файлы.

- [`CLAUDE.md`](./CLAUDE.md) — правила проекта, стек, соглашения и ограничения.
- [`agents.md`](./agents.md) — именованные AI-процессы: роли, входы и выходы.

Источник правды по архитектуре находится в
[`../docs/ARCHITECTURE.md`](../docs/ARCHITECTURE.md).

Порядок чтения для нового участника, человека или AI:

1. `docs/ARCHITECTURE.md` — система, которую мы строим.
2. `.claude/CLAUDE.md` — как мы пишем в ней код.
3. `.claude/agents.md` — как мы используем для этого AI.
